# Primitiv: Element + Respons

Dette dokumentet organiserer respons-primitiver i fire familier.

## Kategorisering-familien
Velg hvordan elementer kategoriseres:

- **Kartlegging** *(velg én tag)*
- **Flervalgstagging** *(velg flere)*
- **Binær** *(ja/nei per element)*

## Vurdering-familien
Velg hvordan elementer vurderes:

- **Skala 1–5**
- **Dot voting**
- **Tommelfingerregel** *(opp/ned)*

## Innspill-familien
Velg hvordan innspill samles inn:

- **Åpent tekstfelt**
- **Strukturert** *(tittel + detaljer)*
- **Anonym vs. navngitt**

## Prioritering-familien
Velg hvordan elementer prioriteres:

- **Rangering** *(dra-og-slipp)*
- **Parvis sammenligning**
